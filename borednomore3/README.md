# BoredNoMore3 (v0.7.0)

BoredNoMore3 is a comprehensive wallpaper and media management suite for Linux.

## Components
- **CLI**: `bnm3d` (Modern), `borednomore3` (Legacy)
- **GUI**: `bnm3d-gui` (Professional), `bnm3` (Legacy)

## Randomization
Note: Randomization features only occur if the `-r` flag is explicitly set when running the backend CLI.

## Installation
Run `make deb` to create a Debian package or `sudo make install` to install directly to `/usr/local`.
